# Xiraac - Home Services Platform

Xiraac is a modern web application for booking home services, built with Next.js, TypeScript, and Tailwind CSS. It provides a platform for users to book various home services like cleaning, electrical work, plumbing, and more.

---

## 🚀 Features

* 🏠 Browse and book home services
* 👤 User authentication and profile management
* 📅 Booking management system
* 📱 Responsive design for all devices
* 💳 Payment system
* 📩 Email notifications using EmailJS
* 🔥 Firebase integration
* 💬 Feedback system
* 👨‍💼 Career/Job section

---

## 🛠 Tech Stack

* **Frontend Framework**: Next.js 13 (App Router)
* **Language**: TypeScript
* **Styling**: Tailwind CSS
* **Backend/Services**: Firebase
* **Email Service**: EmailJS
* **UI Components**: Headless UI
* **Icons**: Heroicons

## 📦 Getting Started

### Prerequisites

* Node.js 16.8 or later
* npm or yarn

---

### Installation

1. Clone the repository:

```bash
git clone https://github.com/TaabishShaikh07/xiraac.git
cd xiraac
```

2. Install dependencies:

```bash
npm install
```

3. Run the development server:

```bash
npm run dev
```

4. Open in browser:

```bash
http://localhost:3000
```

---

## 🔑 Environment Variables

Create a `.env` file and add:

```bash
NEXT_PUBLIC_FIREBASE_API_KEY=your_key
NEXT_PUBLIC_EMAILJS_SERVICE_ID=your_id
```
